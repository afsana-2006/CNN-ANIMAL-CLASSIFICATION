# CNN-ANIMAL-CLASSIFICATION

Animal Classification AI with CNN

A Streamlit-based web application powered by TensorFlow that detects and classifies animal species from images using a trained Convolutional Neural Network (CNN).

🚀 Features
Upload animal images (.jpg, .jpeg, .png)
Real-time species detection
Displays:
Detected animal species
Confidence level
Probability distribution (bar chart)
Handles different image formats (RGB, RGBA, Grayscale)
Clean and responsive UI with side-by-side layout
🧠 Supported Animal Classes

The model is trained to detect the following animals:

Bird 🐦
Cat 🐱
Deer 🦌
Dog 🐶
Frog 🐸
Horse 🐴
🛠️ Tech Stack
Python
Streamlit
TensorFlow / Keras
NumPy
Pillow (PIL)
